部署到 GitHub Pages

如何获取 GitHub Token？

  1. 访问 GitHub Tokens 设置
  2. 点击 Generate new token (classic)
  3. Note 填写 MarkLume Deploy
  4. 勾选 repo 全部权限
  5. 点击生成，立即复制 Token（只显示一次）
