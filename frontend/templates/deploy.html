部署到 GitHub Pages

智能增量部署，只推送变更文件

如何获取 GitHub Token？

  1. 访问 GitHub Tokens 设置
  2. 点击 Generate new token (classic)
  3. Note 填写 MarkLume Deploy
  4. 勾选 repo 全部权限
  5. 点击生成，立即复制 Token（只显示一次）

Token 仅用于本次部署，不会保存

基于 GitHub Discussions 的评论系统，需要在 giscus.app 获取配置信息

如何获取 Giscus 配置？

  1. 确保你的 GitHub 仓库已开启 Discussions 功能
    仓库 → Settings → Features → 勾选 Discussions
  2. 访问 GitHub Apps - giscus，点击 Install 安装到你的仓库
  3. 访问 giscus.app，在 "Configuration" 部分填入你的仓库名
  4. 选择一个 Discussion 分类（推荐 Announcements）
  5. 页面下方会自动生成 仓库 ID 和 分类 ID，复制粘贴到下方对应字段即可
